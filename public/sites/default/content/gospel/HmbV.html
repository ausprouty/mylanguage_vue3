Vaajtswv tsim lub nub, lub hli, cov nubqub hab lub nplajteb.

Vaajtswv tsim tsajtxhu, tej xyoob ntoo, tej roob, ntseg, naagnoog hab txhua yaam tsaj kws muaj sa.

Vaajtswv tsim tuabneeg, hlub tuabneeg, hab muab txhua yaam puab yuav tsum muaj ua rua puab tau lub neej zoo sab.

Tuabneeg tsi noog Vaajtswv lug hab tau moog ua lawv le puab lub sab nyam ua xwb, tsi ua lawv le Vaajtswv has kuas puab ua.

Tom qaab tuabneeg tsiv ntawm Vaajtswv moog lawm, puab nov qaab Vaajtswv hab cuab hawm lwm yaam xws le tej pobzeb, tej xyoob ntoo hab cov ntsujplig phem.

Hab, tuabneeg yuav tau ua dlejnum nyaav heev txha tau noj tau haus.

Tabsis Vaajtswv tseem hlub tuabneeg hab tau khaiv Nwg cov tubkhaiv moog qha rua puab txug txuj xuv zoo has txug Nwg txujkev hlub hab kev zaam txim.

Koj puas paub txug plaub yaam kws Vaajtswv xaav kuas koj paub?

1. Vaajtswv HLUB koj hab xaav muab ib LUB TSWVYIM zoo kawg nkaus rua koj lub neej.

VAAJTSWV TXUJKEV HLUB

“Vaajtswv hlub nplajteb kawg le Nwg muab Nwg tuableeg Tub (Yexu Kheto) kuas txhua tug kws ntseeg Nwg yuav tsi tuag tabsis tau txujsa ntev moog ibtxhis.” Yauhaa 3:16

VAAJTSWV LUB TSWVYIM

Yexu hastas, “Kuv lug kuas mej muaj txujsa, lub neej puvnpo (lub neej kws zoo tshaajplawg).” Yauhaa 10:10

Yog vim le caag coob tug tuabneeg tsi muaj lub neej puvnpo nuav?

Rua qhov…

2. Txhua tug yog tuabneeg txhum. Peb ua tej yaam kws tsi yog hab tej nuav ua rua peb txaav dleb ntawm Vaajtswv. Peb paub tsi tau hab zoo sab tsi tau rua Vaajtswv txujkev hlub hab lub tswvyim rua peb lub neej.

TUABNEEG MUAJ KEV TXHUM

“Txhua tug tuabneeg tau ua txhum.’ (Puab tsi ua neej le kws Vaajtswv has kuas puab ua). Loo 3:23

Vaajtswv tsim tuabneeg nrug Vaajtswv ua phoojywg, tabsis vim peb tsuas xaav ua kuas zoo peb lub sab xwb, peb tau xaiv taug peb txujkev tsi muaj Vaajtswv. Peb txujkev nrug Vaajtswv ua phoojywg tau puastsuaj lawm. Qhov peb taug peb txujkev nuav yog peb ua tej kws Vaajtswv tsi xaav kuas peb ua losyog tsi quavntsej txug Vaajtswv. Vaajlugkub hu qhov nuav hastas yog kev txhum.

TUABNEEG RAUG CAIS DLEB NTAWM VAAJTSWV.

“Tug nqe them kev txhum yog kev tuag.” Loo 6:23

Txujkev tuag nuav yog tshaaj le kws peb lub cev tuag xwb – nwg yog tuag saab ntsujplig hab – kws yog raug cais dleb ntawm Vaajtswv hab ua tsi tau phoojywg nrug Vaajtswv.

Vaajtswv yog tug dlawbhuv hab zoo tabsis tuabneeg yog cov muaj kev txhum. Qhov txawv ntawm Vaajtswv hab tuabneeg luj heev ib yaam le ib lub kwjhaa luj hab tub kws cais ob leeg lawm. Tuabneeg taabmeeg xaav moog cuag Vaajtswv kuas tau lub neej npuvnpo, lub neej zoo puab tug kheej ua, xws le ua tuabneeg zoo, coj lawv le kevcai nplajteb hab ua zoo rua lwm tug. Tabsis tuabneeg yeej moog cuag tsi tau Vaajtswv dlhau lug ntawm puab tug dlaag zug.

3. Yexu Kheto tsuas yog TUAB txujkev kws Vaajtswv muab rua tuabneeg tshem tawm puab tej kev txhum. Dlhau lug ntawm Nwg, koj paub tau hab zoo sab tau rua Vaajtswv txujkev hlub hab lub tswvyim rua koj lub neej.

NWG TUAG NYOB PEB LUB CHAW

“Tabsis Vaajtswv tau qha tas Nwg hlub peb npaum le caag; nwg yog thaum peb tseem yog tuabneeg txhum Yexu Kheto tau tuag rua peb.” Loo 5:8

Sau tug ntoo khaublig, Nwg tau raug txim rua peb cov kev txhum.

NWG TAU SAWV HUV QHOV TUAG ROV LUG

“Yexu Kheto tuag rua peb cov kev txhum…Nwg tau muab log…Nub peb Nwg tau sawv rov lug muaj sa nyob ib yaam le Nwg tau qha rua Nwg cov thwjtim hastas Nwg yuav ua le ntawd…

“…Nwg tshwm rua Petu hab tom qaab rua taagnrho kaum ob tug thwjtim. Ces Nwg tshwm rua ntau tshaaj le kws tsib puas leej cov kws lawv Nwg qaab ua ib zag.” 1 Kauleethau 15:3-6

NWG YOG TUAB TXUJKEV XWB

Yexu Kheto hastas, “Kuv yog Txujkev, Kuv yog Qhovtseeb, Kuv yog Txujsa. Yeej tsi muaj leejtwg moog cuag tau Leejtxiv tsuas yog lug ntawm Kuv xwb.” Yauhaa 14:6

Vaajtswv tau tuam ib tug choj rua lub kwjhaa luj hab tub kws cais peb tawm ntawm Vaajtswv thaum Nwg khaiv Nwg tug Tub, Yexu Kheto, lug tuag rua sau tug ntoo khaublig nyob peb lub chaw.

Paub peb yaam le nuav xwb tseem tsi tau zoo txaus…

4. Peb yuav tau NTSEEG Yexu Kheto ua peb tug Cawmseej hab tug Tswv. Ces peb txha paub tau hab muaj tau kev zoo sab rua Vaajtswv txujkev hlub hab lub tswvyim rua peb lub neej.

PEB YUAV TAU NTSEEG YEXU KHETO

“Tabsis muaj tej txha txais Nwg; ces Nwg muab cai rua puab ua Vaajtswv cov mivnyuas.” Yauhaa 1:12

VAAJTSWV TXUJ KOOBMOOV CAWM PEB DLHAU LUG NTAWM KEV NTSEEG

“Nwg yog Vaajtswv txuj koobmoov mej txha tau dlim, dlhau lug ntawm kev ntseeg. Nwg tsi yog dlhau lug ntawm mej tug dlaag zug. Nwg yog Vaajtswv yaam khoom pub.” Efexau 2:8

PEB NTSEEG YEXU KHETO LUG NTAWM KEV TXAIS NWG LU LUG CAW

Yexu Kheto hastas, “Noog, Kuv sawv ntawm qhovrooj (koj lub sab) hab khob; yog leejtwg nov Kuv lub suab hu hab qheb lub qhovrooj, Kuv yuav lug rua huv nwg (lub sab).” Tshwmsim 3:20

Ntseeg Yexu Kheto yog taug Vaajtswv txujkev tsi yog taug yug txujkev, peb yuav tau ntseeg Yexu rua peb lub neej, ua tug zaam peb tej kev txhum hab ca Nwg hloov peb le kws Nwg pum zoo. Nwg tseem zoo tsi tau txaus qhov kws peb paub hastas tej kws Vaajtswv xaav kuas peb ua yog qhov tseeb es ua rua peb muaj kev zoo sab losyog tu sab thaum kws peb xaav txug tej nuav…

OB LUB VUJ NUAV QHA TXUG OB LUB NEEJ:

Yug losyog kuv yog tug tswj. Kuv yog tug Tswv. Kuv ua txhua yaam kws Kuv xaav ua. Kheto nyob saab nrau kuv lub neej.

Kheto yog tug tswj. Kheto yog tug Tswv. Kuv ua txhua yam kws Nwg has kuas kuv ua. Kuv muaj lub neej txaus sab heev. Kuv yog Vaajtswv ib tug phoojywg.

Taamsim nuav koj lub neej yog lub twg?

Lub neej twg koj xaav tau?

TUG NTSEEG ZAAJ LUG THOV

(Kev thov Vaajtswv yog nrug Vaajtswv sis thaam)

Vaajtswv yeej paub tej kws koj xaav has hab saib koj puas has tseeb tag.

KOJ SIV ZAAJ LUG THOV NUAV LOS TAU.

‘Tswv Yexu, kuv xaav tau koj lug nyob huv kuv lub neej. Kuv ntseeg Koj ua kuv tug Cawmseej hab tug Tswv. Ua tsaug Koj tau zaam kuv tej kev txhum hab muab txujsa ntev rua kuv. Thov Koj tswj kuv lub neej. Thov Koj hloov kuv ua tug tuabneeg kws koj xaav kuas kuv ua. Asmee.

Zaaj lug thov nuav puas yog zaaj koj xaav has rua Vaajtswv?

Yog hastas yog, koj puas kaam has zaaj lug thov nuav taamsim nuav?

TEJ YEXU TAU UA RUA KOJ

Thaum koj ntseeg Yexu ua koj tug Cawmseej hab tug Tswv, koj yeej paub tseeb hastas:

    Kheto lug rua huv koj lub neej (Tshwmsim 3:30; Khaulauxai 1:27)
    Vaajtswv tau zaam koj tej kev txhum lawm (Khaulauxai 1:14)
    Koj yog Vaajtswv ib tug mivnyuam (Yauhaa 1:12)
    Koj tau txujsa ntev (Yauhaa 5:24)
    Koj cuab ua lub neej tshab kws Vaajtswv coj koj moog ua (Yauhaa 10:10)

ZAG NUAV KOJ YUAV TAU:

Moog koom ib pawg-ntseeg kws hawm Yexu Kheto hab qha Vaajtswv Txujlug tseeb.

Koj nyeem Vaajtswv Txujlug txhua nub losyog nrhav ib tug tuabneeg nyeem rua koj.

Yog hastas phoo ntawv nuav paab tau koj, thov koj muab rua lwm tug. Ua le nuav koj paab tau lwm tug lug paub Vaajtswv hab.
